In this assignment we look at a simplified model of a tank with flow in and flow out dynamics that are modeled with a first order differential equation. This is an approximation of the dynamics to get you started on designing a simple Simulink model using parameters for tank height, pipe dimensions etc. that you have previously considered. The following video provides a great description of this system. You are asked to model this system using Simulink and get an understanding of how the tank height changes as you change the parameters of the system. 

At this time - this will be an assignment for each student to develop - so they can get some Simulink experience.  Each person should share their design knowledge with the rest of your team and combine your design recommendations into a final summary that you will submit here as a one-page only pdf document.
